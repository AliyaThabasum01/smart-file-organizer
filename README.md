# 📂 Smart File Organizer

A Python CLI tool that automatically organizes files into folders based on their file types.

## ✨ Features

- Organize Images
- Organize Documents
- Organize Videos
- Organize Music
- Organize Archives
- Move unknown files into an **Others** folder

## 🛠 Tech Stack

- Python
- os
- shutil

## ▶️ Run

```bash
python main.py
```

## 📌 Example

Before:

```
Downloads/

photo.jpg
resume.pdf
song.mp3
movie.mp4
project.zip
```

After:

```
Downloads/

Images/
Documents/
Music/
Videos/
Archives/
Others/
```
